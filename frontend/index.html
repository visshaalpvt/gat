Security Philosophy

HAL 4.0 solves the **Secured String Matching** problem by never allowing the server to see plaintext. We use **Blind Indexing** (HMAC-SHA256) to match search queries against encrypted records. This ensures that even if the database is leaked, no customer information can be recovered without the enterprise-held keys.
